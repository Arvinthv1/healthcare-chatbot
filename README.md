# 🏥 Healthcare Chatbot  

## 📌 About  
The **Healthcare Chatbot** is an AI-powered system built using **Python** and **Machine Learning**, designed to assist patients by providing information about diseases and enabling discussions with doctors. It utilizes a **CSV dataset** containing medical information to offer accurate responses.  

## 🚀 Features  
- ✅ Interactive chatbot for patient queries  
- ✅ Provides disease-related information based on symptoms  
- ✅ Uses **Machine Learning** for intelligent responses  
- ✅ Facilitates doctor-patient discussions  
- ✅ CSV dataset integration for accurate medical data  

## 🛠️ Technologies Used  
- **Python** - Core programming language  
- **Machine Learning (ML)** - Used for chatbot intelligence  
- **CSV Dataset** - Medical data storage
